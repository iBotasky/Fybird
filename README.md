## 第三方框架
| 库                          | 功能             |
| -------------------------- | -------------- |
| **dio**                    | **网络框架**       |
| **flutter_webview_plugin** | **全屏的webview，支持加载本地Html** |
| **flutter_swiper** | **轮播框架** |
| **intl** | **日期格式转换** |



## 运行(先配置好Flutter环境)
clone到本地后删除`pubspec.lock`，然后重新运行即可
